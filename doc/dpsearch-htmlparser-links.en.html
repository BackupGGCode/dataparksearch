4.4. Links

HTML parser understand the following links:

  • <A HREF="xxx">

  • <IMG SRC="xxx">

  • <LINK HREF="xxx">

  • <FRAME SRC="xxx">

  • <AREA HREF="xxx">

  • <BASE HREF="xxx">

    Note: If BASE HREF value has incorrectly formed URL, current one will be used instead to compose relative links.
